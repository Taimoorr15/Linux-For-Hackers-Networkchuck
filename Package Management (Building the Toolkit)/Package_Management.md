apt update && apt upgrade: The "Golden Duo." Refreshes the database and updates software.

apt search [tool]: Find a tool before installing

Example: apt search searchsploit.

dpkg -i [file.deb]: Manual install for Debian packages

git clone [URL]: Grabbing custom exploit code from GitHub.

Example: git clone https://github.com/danielmiessler/SecLists.git (Crucial for wordlists).
