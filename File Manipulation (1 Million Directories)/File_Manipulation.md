mkdir -p path/to/folder: Creates the entire path at once.

Example: mkdir -p hack/targets/company_a/logs.

rm -rf [folder]: Recursive and Forceful deletion.

Warning: Never run rm -rf / as it deletes the entire operating system starting from the root.

touch file{1..10}.txt: Brace expansion.

Example: Creates file1.txt, file2.txt ... file10.txt in one second.
