man [command]: The full manual

Example: man nmap. Use / to search for specific words within the manual (e.g., /stealth)

apropos [keyword]: When you don't know the command name.

Example: apropos "password" will list every tool related to passwords (like passwd, chage, openssl).

whatis [command]: A one-line description of what a tool does.

Example: whatis airmon-ng
